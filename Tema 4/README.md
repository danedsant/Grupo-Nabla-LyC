# 📚 Tema 4 - Análisis Léxico


## 📌 Descripción del Tema 4

Este directorio contiene los códigos fuente, archivos de configuración y la documentación correspondiente al **Tema 4: Análisis Léxico**. En este módulo se aborda la fase inicial del proceso de compilación, explorando desde la teoría fundamental de máquinas abstractas hasta la implementación práctica de escáneres léxicos (lexers) mediante programación manual y el uso de herramientas de metacompilación.


## 📂 Estructura del Directorio

El contenido está dividido en cuatro actividades centrales que combinan investigación teórica y desarrollo de software:

* 📁 **[Actividad 1](./Actividad%201) - Autómatas de Pila:** Análisis teórico y demostraciones visuales sobre el funcionamiento de los autómatas con memoria dinámica (LIFO) y su superioridad frente a los modelos finitos para lenguajes libres de contexto.

* 📁 **[Actividad 2](./Actividad%202) - Lexer para Dockerfiles mediante expresiones regulares en Python:** Implementación  de un analizador léxico robusto en Python. Utiliza jerarquías de expresiones regulares para tokenizar directivas de Docker y gestionar estados de error.

* 📁 **[Actividad 3](./Actividad%203) - Lexer para LRust (Flex / C++):** Diseño formal de un subconjunto del lenguaje de sistemas Rust (*LRust*) y generación automatizada de su escáner léxico utilizando el metacompilador Flex.

* 📁 **[Actividad 4](./Actividad%204) - Flex en Ciberseguridad (Snort):** Análisis técnico sobre la aplicación de motores léxicos basados en Autómatas Finitos Deterministas (AFD) para la tokenización eficiente y en tiempo real de firmas de intrusión en redes de datos.
